# X-ray-series-classification
Classification of a series of x-ray images.
- simpleBinarySeriesClassification.ipynb - binary classification of a series of syntethic images.
- BrainTumorBinarySeriesClassification.ipynb - binary classification of a series of MR brain images with/without a tumor using pretrained Xception network.
- BrainTumorTransformers.ipynb - binary classification of a series of MR brain images with/without a tumor using pretrained transformers.
